# Kanban MVP Delivery Plan

## Phase 1: Scaffolding

Success criteria:
- Next.js app exists in `frontend`
- `.gitignore` covers Node/Next artifacts
- Baseline scripts for `dev`, `build`, `start`, and `lint`

Status: Completed

## Phase 2: Core MVP Features

Success criteria:
- One board only
- Five fixed columns, each renameable
- Card model includes only title and details
- Drag and drop cards across columns
- Add card to any column
- Delete any card
- App starts with dummy data and no persistence/auth

Status: Completed

## Phase 3: UI/UX Polish

Success criteria:
- Color palette matches project requirements
- Professional layout on desktop and mobile
- Visual hierarchy and meaningful motion applied

Status: Completed

## Phase 4: Testing

Success criteria:
- Unit tests for board state logic
- Component tests for core interactions
- Playwright integration test for real browser flow
- Lint and tests passing

Status: Completed
