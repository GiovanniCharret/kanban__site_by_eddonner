# Kanban Project MVP

Single-board Kanban web app built with Next.js.

## Run

```bash
npm run dev
```

## Test

```bash
npm test
npm run test:e2e
```

## Scope

- One board
- Five fixed columns (renameable)
- Cards with title and details only
- Drag and drop cards
- Add and delete cards
- No persistence, auth, archive, or filtering
